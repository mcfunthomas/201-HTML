Future

Jumping Pixel is working on expanding its services by adding web design. One of the next projects will include a solution to achieve some responsiveness for linked and embedded videos. The solution, presented in the Fluid Width Video article, also addresses the <iframe> tag for embedded videos. It needs some CSS tricks that were unable to have been implemented here (self-imposed restrictions). The solution was pioneered by Thierry Koblentz and presented on A List Apart in 2009: Creating Intrinsic Ratios for Video . With this technique, a video can be wrapped in another element which has an intrinsic aspect ratio, then absolute position the video within that. That gives us fluid width with a reasonable height we can count on.
